# PDR Календарь

PWA/web-приложение календаря для планирования рабочих и личных записей.

## Что внутри

- `index.html` — основное приложение
- `manifest.json` — настройки PWA
- `sw.js` — service worker для офлайн-кэша
- `icons/` — иконки приложения

## Как запустить локально

Открой `index.html` в браузере. Для полноценной проверки PWA лучше запустить через локальный сервер:

```bash
python -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000
```

## Как выложить на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи все файлы из этой папки в корень репозитория.
3. В репозитории открой `Settings → Pages`.
4. В `Build and deployment` выбери `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Сохрани настройки.

Через минуту GitHub даст ссылку на веб-приложение.

## Важно

Данные сейчас хранятся в `localStorage` конкретного браузера. Если очистить данные сайта или открыть приложение на другом устройстве, записи не перенесутся автоматически.
