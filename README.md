# PDR Календарь

PWA-календарь для GitHub Pages.

## Как выложить

1. Создай репозиторий на GitHub.
2. Загрузи содержимое этой папки в корень репозитория: `index.html`, `manifest.json`, `sw.js`, `.nojekyll`, `icons/`.
3. Открой `Settings → Pages`.
4. Выбери `Deploy from a branch`, ветка `main`, папка `/root`.
5. Через пару минут сайт будет доступен по адресу:

```text
https://ТВОЙ-НИК.github.io/НАЗВАНИЕ-РЕПОЗИТОРИЯ/
```

## Установка на телефон

Android Chrome: открыть ссылку → меню `⋮` → `Установить приложение`.

iPhone Safari: открыть ссылку → `Поделиться` → `На экран Домой`. На iPhone это всё равно называется добавлением на экран Домой, даже если приложение работает как standalone.

## Важно

После загрузки новой версии на GitHub Pages иногда нужно подождать 1–3 минуты и обновить страницу. Если браузер держит старую версию, открой сайт в режиме инкогнито или очисти данные сайта.
