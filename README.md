PDR Календарь — PWA для Android / GitHub Pages

Файлы нужно загрузить в КОРЕНЬ репозитория:
- index.html
- manifest.webmanifest
- sw.js
- icon.svg
- icons/icon-192.png
- icons/icon-512.png
- .nojekyll

Что исправлено в этой версии:
- скролл месяца на Android: ячейки больше не блокируют вертикальный свайп;
- перетаскивание на телефоне включается долгим удержанием записи, а обычный свайп листает календарь;
- PWA-манифест сделан по схеме, как в рабочем приложении учета доходов;
- service worker стал устойчивым: не падает, если один из файлов не закэшировался.

После загрузки на GitHub Pages:
1. Открой сайт в Chrome на Android.
2. Открой /manifest.webmanifest — должен открыться JSON.
3. Открой /sw.js — должен открыться JS-файл.
4. Если раньше уже открывал старую версию: Chrome → Настройки сайта → Хранилище → очистить данные этого сайта.
5. Закрой вкладку, открой сайт заново и подожди 10–30 секунд.

Если Chrome считает приложение устанавливаемым, вверху появится кнопка “Установить” или пункт в меню Chrome.
